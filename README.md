**Goal:** Transform from failing (8 pts) to comfortably passing (12+ pts)

## Deploy to Railway

1. Push this repo to GitHub
2. Go to [railway.app](https://railway.app)
3. New Project → Deploy from GitHub repo
4. Railway auto-detects the build settings
5. Your app will be live at `your-app.up.railway.app`

## Deploy to Vercel

1. Push to GitHub
2. Import to [vercel.com](https://vercel.com)
3. Framework: Create React App
4. Deploy

## Deploy to Replit

1. Create new Replit with React template
2. Replace src/ with this src/
3. Run

## Embed on wayfindersalmanac.com

Add `?embed=true` to the URL for a minimal progress widget:

```html
<iframe 
  src="https://your-app.up.railway.app?embed=true"
```

## Diet Guidelines (Pescatarian)

- ✓ Fish (tuna, white fish, shrimp, tilapia, cod) — NO SALMON (allergy)
- ✓ Eggs, dairy (yogurt, cheese, cottage cheese)
- ✓ Tofu, tempeh, legumes
- ✓ Protein powder
- ✗ No beef, pork, chicken, turkey, or any meat

**Daily targets:**
- Protein: 195g minimum
- Calories: ~2400 (training days), ~2000 (rest days)
- Water: 1 gallon
- Sleep: 7+ hours

## Program Phases

1. **Phase 1 (Weeks 1-6):** Base Building
   - Zone 2 cardio foundation
   - Heavy lifting maintenance
   - Push-up volume accumulation

2. **Phase 2 (Weeks 7-10):** Sharpening  
   - Increased interval intensity
   - Race-pace work
   - Time trials on weeks 8 & 10

3. **Phase 3 (Weeks 11-12):** Taper
   - Reduced volume
   - Maintained intensity
   - Full rest before test
